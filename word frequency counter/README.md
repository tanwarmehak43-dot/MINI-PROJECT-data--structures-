# Data Structures Mini Projects

This repository contains 4 mini projects based on Python Data Structures such as Lists, Tuples, Dictionaries, Sets, and Comprehensions.

## Projects Included

### 1. Student Result Management
- Store student names and marks in a dictionary
- Calculate average marks
- Find topper
- Display grades

### 2. Contact Book CLI
- Add contacts
- Search contacts
- Delete contacts
- View all contacts

### 3. Shopping Cart System
- Add items to cart
- Remove items
- View cart items
- Calculate total bill
- Track unique categories using sets

### 4. Word Frequency Counter
- Count frequency of each word
- Sort words by frequency
- Display top 5 most frequent words

## Technologies Used
- Python
- Lists
- Dictionaries
- Sets
- Loops
- Conditional Statements

## How to Run

python main.py
